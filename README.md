# Grade Transfer Tool

A simple web-based tool that helps lecturers transfer student marks from a CSV file into an Excel template, matching by student number.

## How to Use

1. Download the code as ZIP or clone the repo.
2. Open `index.html` in any modern browser.
3. Upload your CSV (with marks) and Excel template.
4. Select columns and click "Transfer".
5. Download the updated Excel file.

No installation required. Runs entirely in the browser.
